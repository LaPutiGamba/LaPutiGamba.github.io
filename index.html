SHOOTING ARIA

Click on the button to start the game!

This game is a Space Shooter with diferents enemies, bosses and enhancers.

KEYS

  • Move Left: Left Arrow (<-) or A
  • Move Right: Right Arrow (->) or D
  • Shoot: Space
  • Bomb Enhancer: M

TECHNICAL SPECIFICATIONS

  • Bullet Enhancer: If you pick up this enhancer you will shot one more bullet every time you shoot. There are a maxium of 5 bullets if the points are less than 10.000, 7 if are less than 20.000 and 10 if are more than 20.000. If you achieve the maximum and you pick another enhancer you will receive points.
  • Heart Enhancer: If you have two lifes left you will have the chance to get this enhancer. If you pick it up you will totaly heal.
  • No Bullets Enhancer: If you pick up this enhancer you won't be able to shoot for 5 seconds.
  • Bomb Enhancer: If you pick up this enhancer you will have a bomb that you will be able to explode with the M key. If you explode it all the enemies (bosses no) will die.
  • 10.000 points: If you reach 10.000 points a boss will spawn. When you kill it the ships will change.
  • 20.000 points: If you reach 20.000 points another boss will spawn. When you kill it the ships will change.
  • Change player ship color: If you select one of the colors the ship will change. You can change it in the middle of the game.

TECHNICAL FEATURE EXPLANATION

  • Enhancers: I put a lot on effort in doing the enhancers and I did it by changhing the image of the enemy. I could have did it in another way but for me it seemed the easiest. I changed the image, the width and the height and I also turn on a boolean. Then I started doing like with everything: conditions. If the boolean was true then the bullet when collisions with the player he doesn't get hurt, if the bullets hit the enhancer they didn't kill it, etc.
